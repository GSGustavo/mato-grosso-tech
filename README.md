## Como contribuir

Conhece alguma comunidade de TI do Mato Grosso que ainda não está listada aqui?

1. Faça um **fork** deste repositório
2. Adicione o link na seção correta do `README.md` (ou crie uma nova seção, se necessário)
3. Abra um **Pull Request** com uma breve descrição da comunidade

Também pode abrir uma **Issue** com o link e o nome do grupo que a gente inclui por aqui. 🙌

> **Atenção:** apenas links de grupos públicos e relacionados à área de tecnologia serão aceitos.
